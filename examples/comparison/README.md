# Method Comparison Examples

This directory contains side-by-side comparison samples showing the output of different humanization methods.

## Files

| File | Description |
|------|-------------|
| `original.txt` | Original AI-generated text (input) |
| `method1_translation_chain.txt` | Output from Method 1: Translation Chain |
| `method2_llm_rewrite.txt` | Output from Method 2: LLM Rewriting |
| `method3_detection_guided.txt` | Output from Method 3: Detection-Guided Loop |
| `method4_mixed_engine.txt` | Output from Method 4: Mixed-Engine Translation |
| `lynote_result.txt` | Output from Lynote.ai (all methods combined, auto-selected) |

## How to Read These

Each file processes the same input text. Compare them to understand:
- How each method changes the text differently
- The trade-offs between naturalness and meaning preservation
- Why combining methods intelligently (as Lynote.ai does) produces better results than any single approach

> **Try it yourself:** Paste the original text into [Lynote.ai](https://lynote.ai) and compare the result with the single-method outputs.
